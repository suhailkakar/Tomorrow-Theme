## Installation

1.  Open  **Extensions** sidebar panel in VS Code.  `View â†’ Extensions`
2.  Search for  `Tomorrow`  - find the one by  **Suhail Kakar**
3.  Click  **Install**  to install it.
4.  Code > Preferences > Color Theme >  **Tomorrow**
